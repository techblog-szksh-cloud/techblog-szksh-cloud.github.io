### aqua

:star: Over 200 stars :star:

The notable change is [the support of checksum verification](https://aquaproj.github.io/docs/reference/checksum/).

Please see [the GitHub Project](https://github.com/orgs/aquaproj/projects/8) to check prioritized issues.

- WIP
  - [v1.22.0-1](https://github.com/aquaproj/aqua/releases/tag/v1.22.0-1)
    - :rocket: Support filtering packages with tags
  - [#1306](https://github.com/aquaproj/aqua/issues/1306) Support restricting registries and packages globally for security
- [v1.21.0](https://github.com/aquaproj/aqua/releases/tag/v1.21.0)
  - Support changing `type` in `overrides`

  - cp: Skip creating links
- [v1.20.1](https://github.com/aquaproj/aqua/releases/tag/v1.20.1)
  - update-checksum: Fix the checksum of http package calculated by `-deep` option

  - :tada: Support the checksum verification
  - :tada: cp: Support coping all packages

  - nil check to avoid panic
