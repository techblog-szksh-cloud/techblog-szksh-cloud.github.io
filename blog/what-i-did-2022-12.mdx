## OSS Development

### aqua

The notable change is [Cosign and SLSA Provenance support](https://aquaproj.github.io/docs/reference/cosign-slsa/).
You can install aqua and tools securely.
And the Improvement of `aqua update-checksum` command is also important.

Please see Document and Release Notes for the detail.

- [v1.28.0](https://github.com/aquaproj/aqua/releases/tag/v1.28.0)
  - Improved aqua update-checksum command
    - Ignore unrelated files' checksums in checksum files
    - Support pruning unused checksums in aqua-checksums.json
- [v1.27.0](https://github.com/aquaproj/aqua/releases/tag/v1.27.0)
  - Support a new field `private` for private packages and registries
  - Support template in `go_install` package's `path` attribute
- [v1.26.2](https://github.com/aquaproj/aqua/releases/tag/v1.26.2)
  - Install Cosign asset for correct platform even if `AQUA_GOOS` and `AQUA_GOARCH` are set
- [v1.26.1](https://github.com/aquaproj/aqua/releases/tag/v1.26.1)
  - Skip installing Cosign and verify with Cosign in windows/arm64
  - Check if Cosign is supported
- [v1.26.0](https://github.com/aquaproj/aqua/releases/tag/v1.26.0)
  - Support package verification by Cosign and slsa-verifier
- [v1.25.2](https://github.com/aquaproj/aqua/releases/tag/v1.25.2)
  - generate: fix a bug of `--pin` option
